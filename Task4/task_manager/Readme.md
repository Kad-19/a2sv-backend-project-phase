# Task Manager

A RESTful API for managing tasks, built with Go.

## Features

- Create, read, update, and delete tasks
- JSON-based API
- Simple error handling

## Project Structure

- `main.go`: Entry point
- `controllers/`: Handles HTTP requests
- `models/`: Data models
- `data/`: Business logic/service layer
- `router/`: API routing
- `docs/`: Documentation
